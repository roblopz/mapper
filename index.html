AutoMapper TypeScript

An object-object mapping solution by convention in TypeScript

Get Started

Familiarity

AutoMapper TypeScript follows as close as possible to the original .NET AutoMapper so that it is familiar to use

Well-documented

Documentations are powered by Docusaurus which allows for ease of documenting AutoMapper TypeScript API

Written in TypeScript

Written entirely in TypeScript which allows the consumers to get type-safety out of the box.

Plugin-based

Extensibility is achieved with Plugin-based approach. Official plugins allow for working with TS/ES6 Classes and POJOs. Custom plugins can be created with well-defined API and guide.

NestJS Integration

Official integration with NestJS

Powered by Nx

AutoMapper TypeScript packages are powered by Nx Dev Tools which paves the way for maintainability, scalability, and also welcoming to contributors
